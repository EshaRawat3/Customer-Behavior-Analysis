# Customer-Behavior-Analysis
Delivered an end-to-end customer behavior analytics solution by integrating Python, SQL, and Power BI. Processed and cleaned transactional shopping data using Python (Pandas, NumPy) and structured it in SQL to enable efficient querying and KPI calculation. Performed SQL-based analysis (joins, aggregations, window functions) to uncover purchasing patterns, customer segments, and product-level trends. Built interactive Power BI dashboards with DAX measures to visualize customer frequency, spending behavior, category performance, and time-based trends, enabling data-driven insights into customer preferences and business opportunities.
